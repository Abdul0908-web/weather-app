# 🌤 Weather App

A simple and responsive weather application that displays real-time weather information for any city.

This project was built for learning, experimentation, and portfolio purposes.

---

## 🚀 Features

- Search weather by city name
- Displays temperature and condition of the sky
- Clean and responsive user interface
- Real-time data from weather API

---

## 🛠 Tech Stack

- Python (PyQt5)
- CSS
- Weather API
